## Installation

```
- Install [CSFML-2.5.1](https://www.sfml-dev.org/download/csfml/index-fr.php)
- Git clone (green button)
- Compile: cd /in_the_rpg_repo ; make ; ./my_rpg
```

## Usage

```
./my_rpg -h will display help
```
